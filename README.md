# Bank Management System

A dynamic web-based Bank Management System developed using Java Full Stack technologies.

This application provides banking operations for users and management functionalities for admin. It simulates real-world banking tasks like account creation, deposit, withdrawal, fund transfer, customer management, and transaction monitoring.

---

## 🚀 Technologies Used

- Core Java
- JDBC
- Servlet
- JSP
- MySQL Database
- HTML
- CSS

---

## ✨ Project Features

### 👤 User Module
- User Registration
- User Login
- Open Bank Account
- Deposit Money
- Withdraw Money
- Check Bank Balance
- Transfer Amount
- View Personal Transactions
- Logout

### 👨‍💼 Admin Module
- Admin Login
- View All Customers
- Search Customer by Account Number
- View All Transactions
- Logout

---

## 🗄 Database Connectivity

MySQL is used as backend database for storing:

- user details
- account information
- transaction records

JDBC is used for establishing connectivity between Java application and MySQL.

---

## ▶️ How to Run This Project

1. Clone the repository
2. Import project as Maven Project in Eclipse IDE
3. Configure Apache Tomcat Server
4. Create MySQL database and required tables
5. Run the application on localhost server
